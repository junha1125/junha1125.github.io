## **나의 코드**

```python
def solution(arr):
    start_stack = []
    answer = 0
    
    for i in range(len(arr)):
        if arr[i] == '(':
            start_stack.append(i)
        else:
            diff = i - start_stack.pop()
            if diff == 1:
                answer += len(start_stack)
            else:
                answer += 1
                
    return answer
```
